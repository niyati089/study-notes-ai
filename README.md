Project Overview
Study Notes AI is an AI-powered application that allows users to upload PDFs, generate concise summaries, and interact with document content via natural language queries. The system uses text processing and semantic search to provide context-aware responses.

Features

Upload PDF documents and extract text content

Generate concise summaries of documents

Ask questions and get responses based on document content

Interactive, user-friendly interface

Key Skills / Technologies

Python

Text Processing & Chunking

Natural Language Processing (NLP)

Semantic Search & Embeddings

Interactive Web App Development

Getting Started

Clone the repository:

git clone https://github.com/niyati089/study-notes-ai.git


Install required packages:

pip install -r requirements.txt


Run the app (example using Streamlit):

streamlit run app.py
